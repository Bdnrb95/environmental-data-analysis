# environmental-data-analysis

# Overview
This project looks at environmental monitoring data to track contamination trends and spot periods of high risk.
The goal was to provide clear insights for safety planning and mitigation.

# Tools Used
Python (Pandas, Matplotlib, Seaborn)

# Process
Cleaned and prepared the dataset for analysis  
Performed time-series analysis to find trends over time  
Identified periods with elevated contamination levels  
Visualized results to support recommendations

# Key Takeaways
Certain locations and times show higher contamination risk  
Clear patterns emerge that can inform monitoring schedules  
Insights support proactive mitigation strategies

# Dataset
Simulated environmental data with contamination measurements over time and by location.
